# AGENTS.md

Правила для AI-агентов при работе с этим проектом.

## HTTP запросы

- Использовать `http` (HTTPie) вместо `curl`
- Пример: `http localhost:3000/api/`

## Язык

- Документация на русском языке
- Комментарии в коде на английском

## Структура проекта

- `cmd/` — точка входа (main.go)
- `config/` — конфигурация
- `internal/` — внутренние пакеты
- `docs/learn-go/` — документация по Go
